# F1 Racing History

A modern, responsive static web application showcasing the rich history of Formula 1 racing, featuring:

- **Interactive Timeline**: Journey through F1's most significant milestones from 1950 to today
- **Legendary Champions**: Profiles of the greatest drivers in F1 history
- **Iconic Moments**: Memorable races and achievements that defined the sport
- **Responsive Design**: Optimized for all devices and screen sizes

## Technologies Used

- HTML5
- CSS3 (with custom animations and responsive design)
- Vanilla JavaScript (Intersection Observer API, smooth scrolling)

## Deployment

This site is deployed on Azure Static Web Apps with GitHub integration for continuous deployment.

## Local Development

Simply open `index.html` in your browser to view the site locally.
